## How to run
Installation des librairies utiles :  
```bash
npm install -g truffle
npm install dotenv @openzeppelin/test-helpers @truffle/hdwallet-provider @openzeppelin/contracts
```
  
Exécution des tests unitaires :  
```bash
truffle test
```
